Mini ERP - Expo (JavaScript)
===========================

Bu proje örneği Expo ile çalışacak basit bir mini ERP uygulamasıdır.
İki ekran içerir: Stok Yönetimi ve Satış Yönetimi.

Nasıl çalıştırılır:
1. Bilgisayarınızda Node ve Expo CLI kurulu olmalıdır.
2. Proje kökünde terminalde:
   npm install
   npm start
3. Expo Metro açıldıktan sonra telefonunuzdaki Expo Go ile tarayın veya simulator/emulator başlatın.

Not: Bu zip çevrim içi bağımlılıkları içermez; 'npm install' çalıştırmanız gerekecek.
